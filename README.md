# **Exercícios de Programação em Python**

Este repositório contém uma coleção de exercícios realizados durante a disciplina de **Programação em Python** na faculdade. O foco é explorar a sintaxe da linguagem e desenvolver habilidades de lógica e resolução de problemas com Python.

## **Conceitos Abordados**

- **Estruturas de Controle**: Uso de condicionais (`if`, `else`, `elif`) e laços de repetição (`for`, `while`).
- **Funções**: Criação e utilização de funções para modularizar o código.
- **Estruturas de Dados**: Manipulação de listas, tuplas, dicionários e conjuntos.
- **Orientação a Objetos**: Aplicação de classes e objetos, encapsulamento, herança e polimorfismo em Python.
- **Tratamento de Exceções**: Gerenciamento de erros com `try`, `except`, `finally`.
- **Bibliotecas Padrão**: Uso de bibliotecas como `math`, `random`, entre outras.
